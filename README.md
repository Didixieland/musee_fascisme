# musée sur le fascisme
Un musée sur le fascisme italien en réalité virtuelle, réalisé avec a-frame. L'objectif est de fournir des infos historiques, mais surtout d'amener le spectateur à réfléchir sur ce sujet et l'actualité. Vous vous trouverez face à trois salles avec trois thèmes interdépendants : les origines de la violence, la dictature et le fascisme aujourd'hui.
